# Video-Redux
Proyecto de Video creado en ReactJs con implementación Redux

## Instalación

```
npm install
```

## Uso en Desarrollo
```
npm run build:dev

http://localhost:9000
```

## Uso en Producción
```
npm run build:prod
```
